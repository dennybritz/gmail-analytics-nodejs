Dump your gmail account

On June 26, 2014 Google announced a new Gmail API. Among other things, the API allows developers to read messages, send mail, and manage labels. This little tool can be used to dump your gmail account, or a subset of specific messages from your gmail account.

git clone URL
cd gmail-dump-nodejs
node message-dump-console.js > dump.jsonlines

Dump specific messages

You can specify an optional argument to message-dump-console specifying the Gmail query. The query can be any valid gmail query string described in the documentation.

node message-dump-console.js "before:2013/01/01 after:2012/05/05"
